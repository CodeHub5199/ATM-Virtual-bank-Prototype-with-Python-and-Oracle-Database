# ATM-Virtual-bank-Prototype-with-Python-and-Oracle-Database
ATM-Virtual bank (Prototype) with Python and Oracle Database | Technology used – Python, SQL ORACLE 11G | Modules and Libraries used – cx_Oracle, Pandas, Datetime, time, pygame, etc | Features – New account creation, Login, Withdraw and deposit money, Daily withdrawal limit, Account statement, Forgot PIN, Reset PIN, Delete Account, etc.
